# markuplint-md-parser
Markdown parser for markuplint

## Installation

```bash
$ npm install -D @gyugyu/markuplint-md-parser

$ yarn add -D @gyugyu/markuplint-md-parser

$ pnpm add -D @gyugyu/markuplint-md-parser
```

## Usage

```json
{
  "parser": {
    "\\.md$": "@gyugyu/markuplint-md-parser"
  }
}
```
